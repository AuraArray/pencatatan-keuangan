# 💰 Sistem Pencatatan Keuangan Sederhana

Aplikasi berbasis web sederhana untuk mencatat pemasukan dan pengeluaran harian. Dibuat menggunakan teknologi web dasar untuk keperluan pengelolaan keuangan pribadi secara cepat dan efisien.

## 🚀 Fitur Utama
* **Pencatatan Cepat:** Input keterangan, nominal, dan jenis transaksi (Pemasukan/Pengeluaran).
* **Perhitungan Otomatis:** Saldo akhir akan terhitung secara otomatis setiap kali ada transaksi baru.
* **Penyimpanan Lokal:** Menggunakan **LocalStorage**, sehingga data tetap tersimpan meskipun browser ditutup atau halaman di-*refresh*.
* **Desain Responsif:** Tampilan bersih dan nyaman dilihat di perangkat desktop maupun mobile.
* **Pembersihan Data:** Fitur hapus semua riwayat untuk memulai pencatatan baru.

## 🛠️ Teknologi yang Digunakan
* **HTML5:** Struktur aplikasi.
* **CSS3:** Desain antarmuka modern dan responsif.
* **JavaScript (Vanilla):** Logika perhitungan dan manipulasi DOM.
* **GitHub Pages:** Digunakan untuk hosting agar aplikasi bisa diakses secara online.
